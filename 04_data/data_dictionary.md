# Data Dictionary

This document describes the fields included in the sample dataset used for operational monitoring and reporting.

| Field Name | Description |
|-----------|-------------|
| site_id | Unique identifier for each operational site |
| region | Geographic region where the site operates |
| stage | Current operational stage |
| records_expected | Total number of records expected to be processed |
| records_processed | Number of records processed so far |
| processing_completion_pct | Percentage of processing completed |
| open_incidents | Total number of open incidents |
| critical_incidents | Number of open critical incidents |
| provider_status | Status of external provider compliance |
| last_update | Date of the last data update |
